# 서울한정 상가검색
빅데이터를 활용한 예제를 만들어보았습니다. (서울한정) 특정상가 검색하기
대학교 과제로 만들어봤습니다.

# 적용하기

1. 주피터 노트북 설치
2. data의 있는 소스 붙혀넣기
3. data의 있는 data1.csv 파일 경로확인 및 지정
4. 실행

------------------------
